PlayPauseDemo
=============

This project exists as a demo to explore issues around starting an iOS app that
plays audio, and having it restore a state of "paused" via iOS's *now playing*
functionality, and having the now-playing view (on the lock screen of the phone,
or in a vehicle) correctly show the state as paused, not playing.

To use, build and deploy to an iOS device.  Then launch the app and observe the
playing state on the lock screen.
